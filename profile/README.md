# FreeWrite — PLE project - Thijs Kraaijenbos

This github organisation contains all repositories for **FreeWrite**, a school project developed by Thijs Kraaijenbos.

FreeWrite is a mobile self-help app aimed at young adults (18–27) who are dealing with a mild stimulant addiction and want to cut back or quit. The app uses **narrative therapy** and **cognitive behavioural therapy (CBT)** as its foundation, guiding users through interactive assignments and minigames that help them reflect on and rewrite their personal story.

---

## Repositories

### [FreeWrite-React-Native](https://github.com/PLE-ThijsKraaijenbos/FreeWrite-React-Native)
The **frontend** of the FreeWrite app, built with React Native and TypeScript using Expo. This is the mobile application that users interact with directly.

### [FreeWrite-Django](https://github.com/PLE-ThijsKraaijenbos/FreeWrite-Django)
The **backend** of the FreeWrite app, built with Django REST Framework (DRF). It handles the API, business logic, and data storage for the application.

### [ple-logboek-django](https://github.com/PLE-ThijsKraaijenbos/ple-logboek-django)
A separate Django application used to maintain the **project logbook** for the school assignment. It tracks progress, reflections, and milestones throughout the project.
